= Configuring AsyncAPI specification file
include::_attributes/common-attributes.adoc[]
:context: serverless-logic-configuring-asyncapi

toc::[]

You can consume, trigger, and publish events using an link:https://v2.asyncapi.com/docs/reference/specification/v2.0.0[AsyncAPI] specification file. {ServerlessLogicProductName} uses the link:https://github.com/quarkiverse/quarkus-asyncapi[AsyncAPI extension] for this purpose. This extension generates all necessary link:https://smallrye.io/smallrye-reactive-messaging/smallrye-reactive-messaging/3.3/[Smallrye] properties to enable {ServerlessLogicProductName} to work with the servers specified in the AsyncAPI file automatically. Therefore, you do not need to add any specific properties for your event broker to use AsyncAPI.
